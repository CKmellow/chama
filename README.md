# Chama App

**Chama App** is a mobile application for managing and investing in *chamas* (community savings groups). It allows users to track contributions, manage group activities, and make investments seamlessly.

---

## Features

- **Chama Dashboard:** View and manage your chama groups  
- **Edit Chama Details:** Update group information and settings  
- **Invest in Chamas:** Contribute and track investments easily  
- **YouTube Integration:** Access tutorials and guidance directly in the app  
- **Modern UI:** Clean, responsive, and user-friendly interface  

---

## Tech Stack

- **Frontend:** Kotlin (Android)  
- **Backend:** Node.js  
- **Payments:** M-Pesa Daraja API  
- **Networking:** Retrofit  
- **Video Integration:** YouTube Player API  
- **IDE:** Android Studio

## Getting Started

### a) Download APK
Download the latest debug version of Chama App here:  
[Download Chama App](https://ckmellow.github.io/chama/)

### b) Android Studio Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/CKmellow/chama.git
2. Open Android Studio, select Open an Existing Project, and navigate to the cloned repository.
3. Let Gradle sync and build the project automatically.
4. Connect an Android device via USB or use an emulator.
5. Run the app by clicking Run > Run 'app' or pressing Shift + F10.
6. The app should launch on your device/emulator with the latest updates.
